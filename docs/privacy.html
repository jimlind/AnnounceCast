Privacy Policy

References

Your user id, screen name, and other information accessible to other Discord users or Letterboxd users as well as personal information you pass on via a command usages is referred to as "personal data".

Collection of Data

  • Any personal data that is collected is done in response to your requests or server user requests. Messages are only processed when being asked to do so.
  • Data is only collected to provide services. For these services to be available information about user activity, server identification, and channel identification will be stored.
  • Data can be collected for analytic purposes. This means information such as command usages and errors. We do not collect this data to monitor you as a user. This data is only used to optimize our services and advance the user experience.

Distribution of Data

No user data is being distributed to third parties, it is only used internally.

Deletion of Data

All user-related data can be deleted on request in the support server
